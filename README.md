# devgpython

This is a Python execercice.

## Installation

1. Clone this repository and create a virtual env
    
        git clone https://github.com/xicopetry/devgpython.git
        cd devgpython
        virtualenv dev

2. Install 'mock'
    
        pip install mock

3. Ready to use or test:

        python main.py

    or

        python main.py
